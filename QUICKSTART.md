## ðŸ”§ Troubleshooting

### Images not uploading?
- Check file format (JPG, PNG, WEBP only)
- Ensure file size is reasonable (< 10MB per image)
- Try refreshing the page

### PDF not generating?
- Make sure at least one image is uploaded
- Check browser console for errors
- Try with fewer images first

### Layout looks wrong?
- Adjust margins in settings
- Try different page layouts
- Check image aspect ratios
